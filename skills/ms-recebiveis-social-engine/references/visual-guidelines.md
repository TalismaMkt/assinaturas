# Visual Guidelines — MS Recebíveis

## Estrutura visual validada

Para carrosséis institucionais e educativos da MS:
- faixa vertical escura à esquerda
- área branca limpa à direita
- título forte no bloco esquerdo
- texto de apoio curto
- muito respiro
- sensação de apresentação estratégica
- visual racional e elegante

## Sensação da marca

A peça deve transmitir:
- clareza
- método
- confiança
- leitura de mercado
- maturidade
- organização

## Elementos recorrentes
- tipografia grande
- blocos bem espaçados
- pouco ruído visual
- contraste limpo
- destaque em vermelho quando fizer sentido
- composição institucional, não publicitária exagerada

## Evitar
- cara de template genérico
- excesso de ícones sem função
- poluição visual
- foto aleatória quando o tema for técnico
- estética muito emocional para tema institucional

## Para capas
A capa deve ter:
- headline curta
- subtítulo claro
- leitura mobile forte
- composição simples e premium

## Quando usar pessoas
Só usar quando:
- a peça tiver caráter mais publicitário/comercial
- a referência do usuário apontar para isso
- o assunto pedir humanização mais explícita

Para temas técnicos, preferir layout institucional da marca.
